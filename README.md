An end-to-end data engineering and analytics solution that bridges the gap between static Business Intelligence and real-time Machine Learning applications. This project features a **Power BI Dashboard** for executive insights and a **Streamlit Web App** for operational data management and customer profiling.

## 🌟 Key Features
* **Executive Reporting:** Power BI dashboard identifying a **73% untapped revenue opportunity** in unsubscribed consumers.
* **Real-time Data Exploration:** Streamlit interface with dynamic filters (Season, Category, Age) and live SQL data fetching.
* **ML Profiling Engine:** Predictive model that calculates **Customer Loyalty Tiers** (Standard/Silver/Gold) and segments based on purchase behavior.
* **Transaction Management:** A functional CRUD-style interface to add new transactions directly to a **PostgreSQL** database.
* **DevOps Ready:** Fully containerized environment using **Docker** for seamless deployment.
